## ARC20 list
List all arc20 tokens order by transfer transactions count.

**Request URL**
```
GET /arc20
```

</table>

## ARC20 Transaction list

**Request URL**
```
GET /arc20/:token/txs
```

**Request Parameter**
<table>
    <thead>
        <tr>
            <th>Name</th>
            <th>Default</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td colspan="3">
              <a href="https://github.com/alveyproject/alveyinfo-api/blob/master/README.md#Pagination-Parameters">
                Pagination Parameters
              </a>
            </td>
        </tr>
        <tr>
            <td colspan="3">
              <a href="https://github.com/alveyproject/alveyinfo-api/blob/master/README.md#Block--Timestamp-Filter-Parameters">
                Block / Timestamp Filter Parameters
              </a>
            </td>
        </tr>
        <tr>
            <td>
              <code>reversed</code>
            </td>
            <td>
              <code>true</code>
            </td>
            <td>Return records reversed</td>
        </tr>
    </tbody>
</table>

**Request**
```
GET /arc20/f2033ede578e17fa6231047265010445bca8cf1c/txs?limit=5&offset=0
```
